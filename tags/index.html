Serving a Controller

An instance of Jaguar object represents a server. A Controller can be added to a server using addApiReflected method. serve method serves all the controllers and routes added to the server. Note: We will later see how to add Controller using code generation and how to add route handler functions using Jaguar’s isomorphic muxer. main() async { final server = new Jaguar(); server.addApiReflected(new ExampleApi()); await server.serve(); }

Route annotation: registering a route handler

A route handler is registered with the Controller using one of the route annotations. Get, Post, Delete, Put, etc depending on the HTTP Method the handler should handle. path parameter of route annotation defines the route path. @Api(path: '/api') class ExampleApi { @Get(path: '/hello') String handler(Context ctx) = 'Hello'; }
