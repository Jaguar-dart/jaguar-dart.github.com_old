Route annotation: registering a route handler

A route handler is registered with the Controller using one of the route annotations.

Get, Post, Delete, Put, etc depending on the HTTP Method the handler should handle.

path parameter of route annotation defines the route path.

@Api(path: '/api')
class ExampleApi {
  @Get(path: '/hello')
  String handler(Context ctx) => 'Hello';
}
