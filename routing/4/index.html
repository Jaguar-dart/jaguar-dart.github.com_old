Serving a Controller

An instance of Jaguar object represents a server. A Controller can be added to a server using addApiReflected method.

serve method serves all the controllers and routes added to the server.

Note: We will later see how to add Controller using code generation and how to add route handler functions using Jaguar’s isomorphic muxer.

main() async {
  final server = new Jaguar();
  server.addApiReflected(new ExampleApi());
  await server.serve();
}
